Tags semânticas

Clique aqui para voltar
  1. <section>: É usado para agrupar conteúdo relacionado em uma seção temática. Por exemplo, em um artigo, você pode usar <section> para dividir o conteúdo em seções como introdução, corpo e conclusão.
  2. <article>: É usado para envolver um conteúdo independente e autossuficiente que pode ser distribuído separadamente de outros conteúdos na página. É comumente usado para postagens de blog, artigos de notícias ou histórias completas.
  3. <aside>: É usado para definir um conteúdo relacionado ou complementar ao conteúdo principal da página. Pode conter informações adicionais, anúncios, widgets, entre outros. É geralmente colocado ao lado do conteúdo principal.
  4. <details>: É usado para criar um elemento de detalhes que pode ser expandido ou recolhido para mostrar ou ocultar informações adicionais. É útil quando você deseja fornecer informações extras que podem ser exibidas conforme necessário, como em FAQs ou seções de perguntas e respostas.
  5. <figure>: É usado para envolver uma mídia, como uma imagem ou um vídeo, juntamente com sua legenda usando a tag <figcaption>. É útil quando você deseja associar uma legenda a uma mídia específica.
  6. <figcaption>;: É usado dentro do elemento <figure>; para fornecer uma legenda descritiva para a mídia contida nele. É usado quando você deseja adicionar uma legenda a uma imagem, vídeo ou outro elemento de mídia.
  7. <footer>: É usado para definir o rodapé ou a parte inferior de um documento ou seção. Geralmente contém informações de direitos autorais, links de contato, links para políticas e outras informações adicionais relacionadas ao final da página.
  8. <header>: É usado para definir o cabeçalho ou a parte superior de um documento ou seção. Ele geralmente contém elementos como logotipos, títulos e elementos de navegação.
  9. <main>: É usado para envolver o conteúdo principal de um documento ou seção. Geralmente é único e engloba o conteúdo principal da página.
  10. <mark>: É usado para destacar ou realçar um trecho de texto dentro de um parágrafo ou outro elemento de texto. Pode ser usado para chamar a atenção do leitor para informações importantes.
  11. <nav>: É usado para definir uma seção de navegação, como menus de navegação, links de página ou qualquer outra forma de navegação do site. É comumente usado para criar barras de navegação.
  12. <summary>: É usado em conjunto com o elemento <details> e fornece um título ou resumo para o conteúdo oculto. Geralmente é clicável para expandir ou recolher os detalhes.
  13. <time>: É usado para representar datas e horários. É útil quando você deseja exibir informações de data e hora de forma estruturada, como datas de publicação, eventos, etc.
