Select, Datalist, Textarea - Exemplo

select

<select name="" id=""></select>

É utilizado junto com as tags <option value="PE">Pernambuco</option> e <optgroup label="Times do Norte"></optgroup>

Ex:

    <label for="times">Seu time favorito:</label>
    <select name="time" id="times" required>
        <option disabled selected>---Escolha seu time de coração---</option>
        <optgroup label="Times são paulinos">
            <option value="SP">São Paulo</option>
            <option value="PAL">Palmeiras</option>
            <option value="SAN">Santos</option>
        </optgroup>
        <optgroup label="Times cariocas">
            <option value="FLA">Flamengo</option>
            <option value="FLU">Fluminense</option>
            <option value="VAS">Vasco</option>
        </optgroup>
        <optgroup label="Times nordestinos">
            <option value="FOR">Fortaleza</option>
            <option value="SPOR">Sport</option>
            <option value="BAH">Bahia</option>
        </optgroup>
    </select>

datalist

Ela possui as mesmas tags e atributos utilizados no select. É necessário de um input de texto. Para fazer a ligação é necessário usar o atributo list com o valor do id da datalist dentro da tag: <input type="text" name="texto" id="texto" list="lista"> - <datalist id="lista"></datalist>

Ex:

    <label for="csalgados">Salgados: </label>
    <input type="text" name="salgado" id="csalgados" list="lista-de-salgados">
    <datalist id="lista-de-salgados">
        <optgroup label="Padaria">
            <option value="Pastel">Pastel</option>
            <option value="Pão-pizza">Pão-pizza</option>
            <option value="Empadona">Empadona</option>
        </optgroup>
        <optgroup label="Festa">
            <option value="Rabo-de-tatu">Rabo-de-tatu
            <option value="Coxinha">Coxinha</option>
            <option value="Cachorro-quente">Cachorro-quente</option>
        </optgroup>
    </datalist>

textarea

<textarea name="" id="" cols="30" rows="10">

Utiliza-se os atributos cols="" e rows="" para se definir o tamanho da área (mesmo definindo o tamanho, ela não hávera limite de texto a se escrever). Para definir-se o total de caracteres a ser escrito usamos o atributo maxlength=""

Ex:

    <textarea name="area" id="areatexto" cols="30" rows="10" maxlength="20" placeholder="Digite aqui!!!!">
