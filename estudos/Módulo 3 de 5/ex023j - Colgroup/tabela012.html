AGRUPANDO COLUNAS:

Na parte de HTML:

        Ex:
            <table>
                <colgroup>
                    <col class="cnome">
                    <col class="c s e x o">
                    <col class="cidade">
                    <col class="cprof">
                </colgroup>
            </table>

Na parte de CSS:

        Ex:
            col.cnome {
                    background-color: yellow;
                }
            col.c s e x o {
                    background-color: green;
                }

Para grupo de colunas:

HTML:

    Ex:
        <colgroup>
            <col class="cnome">
            <col class="cgrupo" span="2">	(2 colunas serão formatadas)
            <col class="cprof">
        </colgroup>

CSS:

    Ex:
        col.cgrupo {
                background-color: red;
            }
