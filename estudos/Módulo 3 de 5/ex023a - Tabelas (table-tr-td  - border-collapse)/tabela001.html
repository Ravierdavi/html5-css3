Novas tags: <table> - <tr> - <td>

Hierarquia de Tabelas (simples): TABLE = tabela = <table> - TABLE ROW = linha de tabela = <tr> - TABLE HEADER = cabeçalho da tabela = <th> e TABLE DATA = dado de tabela = <td>

A propriedade border-collapse em CSS é usada para definir as bordas da célula presente dentro da tabela e diz se essas células compartilharão uma borda comum ou não.

    <table>
        <tr>
            <td>A1</td>
            <td>B1</td>
            <td>C1</td>
        </tr>
        <tr>
            <td>A2</td>
            <td>B2</td>
            <td>C2</td>
        </tr>
        <tr>
            <td>A3</td>

        </tr>
        <tr>
            <td>A4</td>

    </table>
